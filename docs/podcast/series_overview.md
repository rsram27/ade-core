# The Autonomous Data Engineer — Video Series

## Overview

A 20-episode series showing real agentic data engineering workflows using ADE + Claude Code.

**Channel**: The Autonomous Data Engineer (YouTube)
**Target**: Data engineers, platform engineers, AI practitioners
**Language**: English
**Format**: Screen recording + facecam (8-15 min episodes)

## Episode Structure

### Block 1: Hook + Foundations (Episodes 1-4)

| # | Title | Focus |
|---|-------|-------|
| 1 | "I Run 3 AI Agents Every Morning Before Coffee" | End-to-end demo, wow factor |
| 2 | "What is Agentic Data Engineering?" | Vision, why current tools aren't enough |
| 3 | "The Missing Layer: Why AI Agents Need a Knowledge Graph" | ADE's context system |
| 4 | "My Stack: ADE + Claude Code" | Practical setup, architecture |
