# Recording Checklist — The Autonomous Data Engineer

## Pre-Recording Setup

### Environment
- [ ] Close all unnecessary applications
- [ ] Clean desktop — only essential icons
- [ ] Set terminal font size to 16pt minimum
- [ ] Enable dark mode across all tools
- [ ] Disable notifications (Focus mode on)
- [ ] Close Slack, Teams, email

### Demo Environment
- [ ] Verify demo Databricks workspace is ready
- [ ] Verify demo Power BI model is loaded
- [ ] Verify demo data is synthetic (no real client data!)
- [ ] Pre-run all workflows to know timing
- [ ] Clear terminal history

### Hardware
- [ ] Camera positioned, lighting good
- [ ] Microphone tested, levels good
- [ ] Second monitor for notes (not visible in recording)
- [ ] Headphones ready for audio monitoring

### Software
- [ ] OBS configured with scenes:
  - Full camera (intro/outro)
  - Screen + small facecam (main recording)
  - Screen only (optional for specific moments)
- [ ] Test recording — verify audio/video sync
- [ ] Enough disk space for recording

## During Recording

### Intro (Full Camera)
- [ ] Look at camera, not screen
- [ ] Speak clearly, moderate pace
- [ ] Energy up — this is the hook
- [ ] Keep under 40 seconds

### Screen Recording
- [ ] Narrate what you're doing
- [ ] Pause briefly at key moments (easier to edit)
- [ ] Zoom will be added in post — don't worry about it now
- [ ] If you make a mistake, pause and redo that section

### Outro (Full Camera)
- [ ] Summarize what was shown
- [ ] Clear call to action (subscribe)
- [ ] Mention next episode topic
- [ ] Mention ADE is open source + link in description

## Post-Recording

### Editing (CapCut)
- [ ] Import footage
- [ ] Cut dead time and mistakes
- [ ] Add zooms on key moments (4-5 per video max)
- [ ] Speed up agent "thinking" time (2x-4x)
- [ ] Add background music (screen recording only)
- [ ] Add transitions between terminals
- [ ] Export at 1080p minimum

### Thumbnail
- [ ] Create in Canva or similar
- [ ] Show terminal or workflow
- [ ] Text overlay with episode hook
- [ ] Your face visible
- [ ] High contrast colors

### Upload
- [ ] Title: clear, searchable
- [ ] Description: summary + links + timestamps
- [ ] Tags: agentic, data engineering, Claude Code, Databricks, etc.
- [ ] Thumbnail uploaded
- [ ] End screen added
- [ ] Cards added (link to ADE repo, previous episodes)

### LinkedIn Clips
- [ ] Identify 2-3 best 60-90 second moments
- [ ] Export as vertical (9:16) or square (1:1)
- [ ] Add captions
- [ ] Write LinkedIn post with insight/hook
- [ ] Upload native (not YouTube link)

## Safety Checklist

**Before publishing, verify:**
- [ ] No real client names visible
- [ ] No real project names visible
- [ ] No credentials visible (API keys, tokens, passwords)
- [ ] No real data schemas or table names
- [ ] No Accenture references
- [ ] Demo environment looks realistic but is 100% synthetic
